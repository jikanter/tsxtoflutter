# tsxtoflutter - a scalable translation layer between typescript and flutter components

## Constraints

### Hard
- Output components must compile for iOS, Android, and Web.
- Platform-specific extensions *must* be replicated across iOS, Android, and Web.
- Validate all phases with the `showboat` tool (read `showboat --help` to learn more)
- Use the test-driven development skill for any development work.

### Soft
- Attempt to replicate functionality observed in the TSX model verbatim, but if not possible re-implementation is ok.

### Ask First
- Introduction of new technologies or tools


## Deliverable
A feature-complete transformation layer between TSX components and flutter
